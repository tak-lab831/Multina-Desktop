# 電気泳動結果の判定補助アプリ (Electrophoresis Result Judge Assistant) ver.1.1

電気泳動の解析データ（CSVファイル）を読み込み、サイズ（bpまたはnt）が指定したしきい値（差分）以内に収まっている酷似した試料（ウェル）を自動でグループ化・抽出するGUIアプリケーションです。

---

## 📌 概要

DNA/RNAの電気泳動データから、サイズが近いと思われる試料同士を自動でグルーピングし、確認作業を効率化します。
また、周囲に似たサイズの試料が存在しない「孤立した試料」も同時にリストアップします。

### 主な機能
* **CSVデータのインポート**: 電気泳動装置などから出力された解析データを読み込み。
* **フィルタリング**: `1500 bp/nt` 以下のデータのみを対象に自動フィルタリング（不要なラダーや特定マークの除外）。
* **しきい値設定**: 1〜30の範囲で、近傍判定とするサイズ（bp/nt）の差分を設定可能。
* **グループ表示**: サイズが近い試料の組み合わせをカンマ区切りでリスト化。選択すると該当する詳細データをプレビュー。
* **孤立試料の抽出**: 近い値を持つ試料が他に存在しないウェルを個別に一覧表示。

## 📖 使い方
CSV選択: 「選 択」ボタンを押し、電気泳動結果のCSVファイル（Shift_JIS形式）を選択します。

しきい値設定: 判定基準とする差分の値を入力し、「OK」ボタンを押します（初期値: 20）。

データ処理実行: 「Input Data」ボタンを押すと解析がスタートします。

結果の確認:

左側パネル: 近い値を持つ試料が存在しないウェルが一覧表示されます。

右側上パネル: 近い値を持つ試料同士のグループがリスト表示されます。

右側下パネル: 上パネルで選択したグループに含まれるウェルの詳細なサイズを確認できます。

リセット / 終了: 別のファイルを読み込む場合は「リセット」を、アプリを閉じる場合は「終 了」を押します。

## 📊 入力CSVファイルのフォーマット（想定）
CSVファイルは、ヘッダー行を含み、最低限以下の列名が存在している必要があります。

ウェル名: 試料の場所（例: A1, B2 など）。※名前に「X」が含まれる行（ラダー等）や、サイズが「-」の行は自動で除外されます。

サイズ(bp)/(nt): 各バンドのサイズ数値。
